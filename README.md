# Informationen
Mittels eigner Programmiersprache (DuckScript), kann eine Ente innerhalb eines zweidimensionalem 12x12 Feld gesteuert werden. Alle Befehle sind auf Deutsch und intuitiv zu verstehen, wie z.B. "vorwärtslaufen".
Das Programm simuliert Elemente einer Registermaschine. Das Ausführen von einem DuckScript wird mittels Befehlszeiger bewerkstelligt. Dadurch wir das Skript Zeile für Zeile abgearbeitet. Durch Befehle wie "springen" kann der Befehlszeiger manuell während des Skripts überschrieben werden.
Somit können einfache "if-Abfragen" oder Schleifen durch "springewenn" oder "springenichtwenn" umgesetzt werden.
Außerdem enthält das Programm 20 Register, wo ganze (auch negative) Ganzzahlen gespeichert und später im Skript verwendet werden können.
